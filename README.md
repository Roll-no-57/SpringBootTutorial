## Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/SpringBootTutorial.git
    ```
2. Navigate to the project directory:
    ```bash
    cd SpringBootTutorial
    ```
3. Build the project using Maven:
    ```bash
    mvn clean install
    ```
